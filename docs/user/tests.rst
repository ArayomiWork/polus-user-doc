Испытания
=========

Для определённых видов работ в строительстве необходимо проводить лабораторные или геодезические испытания.
В нашем приложении предусмотрена отдельная форма для такого вида работ, чтобы её создать, перейдите в раздел "Испытания" в главном меню.
