Первоначальная настройка организации
====================================

Первоначальная настройка организации выполняется администратором системы в панели администрирования. 
В дальнейшем панель администрирования используется для подключения организаций-контрагентов и новых пользователей.

Администратор обладает всеми правами настройки.
