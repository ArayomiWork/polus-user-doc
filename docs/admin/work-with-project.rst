Работа с проектом
=================

В этом разделе идёт работа с панелью пользователя, поэтому вам стоит перейти в неё.
О том, как начать с ней работать вы можете почитать в разделе :doc:`/user/installing-and-login`.

Большая часть работы с проектом была описана в пользовательском разделе :doc:`/user/work-with-project`.
Здесь будут описаны только те части, которые пересекаются с работой администратора.

..  note:: Для любого пользователя приложения "СтройКонтроль" мы рекомендуем ознакомится со всей документацией.

    Поэтому, даже если вы администратор, то вам в первую очередь стоит прочесть раздел :doc:`/user`.

..  attention:: Все разделы внутри проекта видны только администратору системы.
    В панели администрирования можно ограничить права участников на проекте.

    Например: предоставить право сотруднику фирмы, который выполняет работы по монтажу перекрытий,
    работать только с профильной документацией и работами в разделах «Проекты», «Документация», «Замечания».

    При этом закрыть ему доступ к документации по всему проекту, разделам «Аналитика», «Участники», «Настройки».

Первоначальные настройки проекта
--------------------------------

В разделе пойдёт речь о проектах, которые вы создавали на предыдущем этапе во время первоначальной настройки организации.

Перед тем, как начать полноценную работу с проектами, не забудьте настроить каждый в разделе "Настройки".
Подробнее об этом написано здесь: :doc:`/user/settings`.
